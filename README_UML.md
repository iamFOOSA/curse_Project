# Диаграмма классов в CLion

## Способ 1: Использование PlantUML плагина

1. **Установите плагин PlantUML в CLion:**
   - File → Settings → Plugins
   - Найдите "PlantUML" и установите
   - Перезапустите CLion

2. **Откройте файл `class_diagram.puml`:**
   - CLion автоматически распознает файл
   - Правой кнопкой по файлу → "PlantUML Preview"
   - Или используйте сочетание клавиш (обычно Alt+D)

3. **Экспорт диаграммы:**
   - В окне предпросмотра: правой кнопкой → Export → PNG/SVG

## Способ 2: Автоматическая генерация из кода (встроенная функция CLion)

CLion может автоматически генерировать диаграммы классов из вашего C++ кода:

1. **Откройте любой заголовочный файл (.h)**
2. **Правой кнопкой мыши на классе**
3. **Выберите "Show Diagram" или "Diagrams" → "Show Class Diagram"**
4. **Или используйте:**
   - Code → Show Diagram
   - Или Ctrl+Alt+U (Cmd+Alt+U на Mac)

5. **Для создания общей диаграммы:**
   - Выделите несколько файлов в Project View
   - Правой кнопкой → Diagrams → Show Diagram

Используйте **Способ 2** (встроенная функция CLion) - это самый простой и точный способ, так как диаграмма будет автоматически синхронизироваться с вашим кодом.

Для красивого документа используйте **Способ 1** с файлом `class_diagram.puml`.
